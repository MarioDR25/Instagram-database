# Instagram Database 

Simple Instagram-style relational database built with C#, Entity Framework Core, and SQLite.


## Database Diagram

![Database Diagram](./images/database-diagram.png)
## Features

- Users
- Posts
- Comments
- Media files
- Followers system
- Self-referencing relationships
- Fluent API configuration
- Composite keys
- SQLite database

## Tech Stack

- C#
- .NET
- Entity Framework Core
- SQLite
